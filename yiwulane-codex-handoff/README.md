# YIWULANE — Codex Website Handoff

This repository is the implementation brief for the YIWULANE marketing website.

## Business

YIWULANE is a Yiwu-based sourcing and hybrid fulfillment partner for growing overseas e-commerce sellers. The offer combines low-MOQ sourcing, supplier management, quality control, branded packaging, China fulfillment, and China + local warehouse scaling.

## Primary conversion

**Start a 3-SKU Pilot**

The website is a B2B lead-generation site, not a product marketplace.

## Recommended workflow in Codex

1. Read `AGENTS.md`.
2. Read every file in `/docs`.
3. Open `CODEX_TASK.md`.
4. Start in Plan mode and produce an implementation plan.
5. Implement the site in small, verifiable phases.
6. Run linting, type checks, tests, and a production build.
7. Review all pages at mobile, tablet, and desktop widths.
8. Return a completion report listing:
   - routes built;
   - integrations requiring credentials;
   - test/build results;
   - remaining content or legal approvals;
   - launch instructions.

## Expected stack

- Next.js 16 App Router
- TypeScript in strict mode
- Tailwind CSS 4
- Vercel-ready deployment
- Data-driven page content
- Server-validated lead form
- No CMS in version 1
